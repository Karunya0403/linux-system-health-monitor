# Linux System Health Monitoring Tool

A Bash-based monitoring utility that provides real-time system information and automated report generation.

## Features

- CPU Usage Monitoring
- Memory Usage Monitoring
- Disk Usage Monitoring
- Top 5 CPU Processes
- Timestamped Report Generation
- List Saved Reports
- View Latest Report
- Menu-Driven Interface

## Technologies Used

- Bash Scripting
- Linux Commands (top, free, df, ps, date)

## Run

chmod +x system_monitor_final.sh
./system_monitor_final.sh
